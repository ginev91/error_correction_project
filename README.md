# Analysis of Error-Correcting Codes in Telecommunications

**Topic:** Linear Algebra and Probabilistic Modeling of Hamming (7,4)

---

## Abstract

This project looks at the mathematical ideas behind keeping data correct when it is affected by noise. It starts from simple parity checking and moves toward the Hamming (7,4) code, and then briefly explores more modern approaches.

Using Python in a Jupyter Notebook, the project shows how matrix operations over finite fields ($GF(2)$) and basic probability can be used to detect and correct errors automatically. The goal is to understand how these mathematical tools work together, not just in theory, but also through simulation.

---

## Getting Started (Anaconda & Jupyter)

### 1. Prerequisites

Make sure you have **Anaconda** or **Miniconda** installed.

This project uses common data science libraries along with `ipywidgets` for interactive simulations.

---

### 2. Environment Setup

Open your **Anaconda Prompt** (Windows) or **Terminal** (macOS/Linux) and run:

```bash
# Install required libraries
conda install numpy matplotlib scipy ipywidgets
```
### 3. Running the Project

After installing the dependencies, launch Jupyter Notebook or JupyterLab:

```bash
jupyter notebook
```
